About Lojban

.i ma cu la Lojban - What is Lojban?

Lojban is a constructed language. This means that it is no different from any other language, except that it was designed specifically and intentionally.

Lojban was made with logic and cross-nationality in mind. A Lojban sentence's grammar is unambiguous.

At the same time, a word or phrase can be of ambiguous meaning, and relative to the context of the situation

Lojban is not 'the perfect language', and has little practical use, but it is fun to learn and use.

This website was not made to teach Lojban. If you want to start learning, Lojban.org is a great resource.
